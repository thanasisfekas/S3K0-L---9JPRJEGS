Οδηγίες για τον "έλεγχο" του use case 7

1. Εκτελέστε το αρχείο memberSearchController.py .
2. Εισαγάγετε τα στοιχεία αναζήτησης και πατήστε το κουμπί "Search".
3. Επιλέξτε το άτομο που εμφανίζεται στα αποτελέσματα.
4. Ελέξτε τα στοιχεία που σας εμφανίζονται -> Μόλις βεβαιωθείτε ότι είναι τα σωστά, πατήστε Create Shift.
  Note: Αυτή η οθόνη πρέπει να είναι σε μεγέθυνση για να φαίνονται σωστά τα στοιχεία.
5. Συμπλήρωση Φόρμας
     -> Ημερομηνία: ΜΟΝΟ μελλοντικές τιμές
     -> Ώρες: Έχουν format: Morning:{start_time:06:00, end_time:14:00}
                        Afternoon:{start_time:14:00, end_time:22:00}
                        Night:{start_time:22:00, end_time:06:00}
6. Μόλις ολοκληρώσετε την συμπλήρωση, πατήστε 'Save Shift'.
