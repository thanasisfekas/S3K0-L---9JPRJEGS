Οδηγίες για τον "έλεγχο" του use case 8

1. Εκτελέστε το αρχείο searchPatientController.py .
2. Εισάγετε τα στοιχεία αναζήτησης και πατήστε "Search".
3. Στην επόμενη οθόνη που θα εμφανιστεί πατήστε "Next", εφόσον έχετε ελέγξει τα στοιχεία του ασθενή.
4. Στη συνέχεια, πατήστε τη συνταγογράφηση που επιθυμείτε.
5. Πατήστε το φάρμακο που θέλετε να χορηγήσετε.
6. Εισάγετε την ποσότητα που θέλετε και πατήστε "Confirm".
   Note: Το πρόγραμμα δεν θα σας αφήσει να επιλέξετε 2 φορές το φάρμακο αν έχετε εισάγει την ποσότητα και το υποβάλατε.
7. Όταν επιλέξετε για όλα τα φάρμακα ποσότητα τότε το πρόγραμμα θα σας επιστρέψει μόνο του στην Οθόνη με τις συνταγογραφήσεις του ασθενή.
